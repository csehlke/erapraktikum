TUM ERA-52 Assembler Projekt (SoSe 2015)
========================================

Build and Test:

    make build
    make test

Requirements
------------

- Linux System (Ubuntu, Debian, Arch, ...)
- `libc6-dev-i386` package (`<sys/cdefs.h>` for 64-Bit Systems only)
